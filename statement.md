# Project Statement

## Project Title
Smart Hospital Appointment System

## Problem Statement
Managing appointments in hospitals can be chaotic without a simple scheduling system. Patients and doctors need a straightforward interface to create and view appointments and avoid double-booking.

## Scope
- Patient management
- Doctor management
- Booking and cancelling appointments
- View appointments and basic persistence

## Target Users
- Patients
- Hospital receptionists / admin staff
- Doctors (to check schedule)

## High-level Features
- CRUD-lite for patients and doctors
- Appointment booking with availability checks
- CSV-based persistence for easy submission
